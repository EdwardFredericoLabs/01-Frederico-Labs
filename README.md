# 01-Frederico-Labs# Olá, eu sou o Edward! 👋
### 🧠 Entusiasta de Sistemas Embarcados, Automação e IA Aplicada ao Hardware

Meu GitHub é o reflexo da minha bancada: um laboratório vivo onde protótipos saem do papel, microcontroladores ganham voz e Inteligências Artificiais deixam de ser apenas teoria para se tornarem parte ativa de soluções reais do mundo físico.

---

### 🛠️ O que eu faço
* **Desenvolvimento de Protótipos:** Do primeiro esboço no papel até o firmware rodando de forma otimizada no hardware.
* **IA Embarcada (Edge AI):** Integração de modelos de linguagem (LLMs) com sensores, displays e atuadores em tempo real.
* **Documentação Transparente:** Projetos registrados com clareza, organização e foco em tecnologia compreensível.

### 🧪 Protótipos em Destaque na Bancada

* **🤖 Assistente Pessoal com IA:** ESP32 + display ILI9341 + Gemini/Groq/OpenRouter, com fallback automático e interface web.
* **🛡️ Frederico Labs Security Platform (FLSP):** Controle de acesso com token criptográfico, biometria via smartphone e log em tempo real.
* **🖼️ Slideshow com SD Card:** Sistema dual-core com pipeline de imagens e leitura dinâmica de arquivos BMP.
* **✈️ Horizonte Artificial:** Instrumento de voo simulado com atitude e telemetria rodando em tela TFT.
